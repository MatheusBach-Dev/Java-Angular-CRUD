Java + Spring Boot, Angular, and PostgreSQL project for learning these languages and frameworks.

How to run the project:

#Clone the repository

git clone https://github.com/MatheusBach-Dev/Java-Angular-CRUD.git

#Initializing the backend

cd backend
mvn spring-boot:run
The backend will start on: http://localhost:8080

#Initializing the frontend

cd frontend
ng serve
The frontend will start on: http://localhost:4200
